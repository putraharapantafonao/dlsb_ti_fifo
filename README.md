# DLSB-TI FIFO  
**Digitalisasi Layanan Sirkulasi Barang Administrasi Jurusan Teknik Informatika Berbasis Web Menggunakan Metode FIFO**

Project ini merupakan implementasi dari tugas mata kuliah **Rekayasa Perangkat Lunak** di Jurusan Teknik Informatika Universitas Malikussaleh.  
Sistem dibangun untuk mendigitalisasi proses **peminjaman dan pengembalian barang inventaris** (misalnya proyektor, kabel HDMI, dan Cok Sambung dengan menerapkan metode **FIFO (First In, First Out)**.
